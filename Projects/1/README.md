SmartTrainStation – MATLAB Online 3D Subway Simulation
======================================================

This project simulates a smart subway train station using MATLAB Online.
It uses basic 3D graphics, animations, schedules, and user interaction.

No VRML, no Simulink, no external toolboxes.

------------------------------------------------------
How to Run
------------------------------------------------------

1. Open MATLAB Online
2. Upload the SmartTrainStation folder
3. Open main.mlx
4. Click Run

------------------------------------------------------
Controls
------------------------------------------------------

W – Move forward
S – Move backward
A – Move left
D – Move right
Mouse click – Interact with displays / ticket machine

------------------------------------------------------
Folder Structure
------------------------------------------------------

vrml/           : 3D environment (MATLAB-based)
scripts/        : Animation, systems, interaction
data/           : Train schedules
main.mlx        : Entry point
